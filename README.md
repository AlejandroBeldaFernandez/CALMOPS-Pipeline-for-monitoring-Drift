# CALMOPS – Pipeline for Monitoring Drift

Pipeline de monitorización de *data drift* y *model drift* con:
- Detectores univariantes y multivariantes (Frouros)
- Comparativa previous vs current (degradación relativa)
- Modos de *retrain* (full, incremental, ventana, stacking, replay mix, recalibración)
- Dashboard en Streamlit
